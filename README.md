# API Testing Project - ReqRes

## Overview
This project contains manual API testing using Postman on ReqRes API.

## Tools Used
- Postman
- ReqRes API

## Test Coverage
- Authentication (Login)
- Users endpoints (GET, POST, PATCH, DELETE)
- Negative test scenarios
- Status code validation
- Response time checks

## How to Run

1. Download the collection file from this repository
2. Import it into Postman
3. Set environment variable:
   baseURL = https://reqres.in/api
4. Run collection using Collection Runner
